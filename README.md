## Installation

```
git clone https://git.sr.ht/~busykoala/py_quantum_computing
cd py_quantum_computing
<path/to/python3.8> -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run Jupyter Notebook

```
cd ./py_quantum_computing
jupyter notebook
```
